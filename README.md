## üöÄ Project Overview

This portfolio represents a convergence of modern web engineering and creative design. It moves beyond static layouts to offer a dynamic user experience, featuring fluid simulations, reactive typography, and 3D interactions. It serves as a comprehensive showcase of my work in **Software Engineering**, **Web Development**, and **Creative Coding**.

## üé® Features & React Bits

This project heavily utilizes **[React Bits](https://reactbits.dev/)** and custom implementations for its unique visual flair.

### üåå Background Effects
*   **Liquid Ether:** A high-performance WebGL fluid simulation used as the global background.
*   **Grid Scan:** A retro-futuristic scanning grid effect overlay.
*   **Click Spark:** Interactive particle explosions on mouse clicks.

### üî† Text Animations
*   **Text Pressure:** Interactive typography that reacts to mouse proximity (variable font weight/width).
*   **Scroll Float:** GSAP-powered character reveal animation triggered by scrolling.
*   **True Focus:** A focus blur effect that highlights specific words.
*   **Variable Proximity:** Text that transforms based on cursor distance.
*   **Shiny Text:** Shimmering text effects for headers.
*   **Scrambled Text:** Matrix-style character decoding effect.
*   **Rotating Text:** Vertical sliding text for role/skill cycling.
*   **Blur Text:** Smooth gaussian blur reveal for paragraphs.

### üß© Interactive Components
*   **Tilted Card:** 3D tilt effect on hover for artist/project cards.
*   **Dome Gallery:** A 3D spherical gallery for photography.
*   **Chroma Grid:** A responsive grid with chromatic aberration effects.
*   **Spotlight Card:** Cards with a moving spotlight gradient border.
*   **Gooey Nav:** A liquid-morphing navigation bar.
*   **Dock:** macOS-style floating dock for mobile navigation.
*   **Logo Loop:** Infinite marquee scrolling for tech stack logos.

## ‚öôÔ∏è Setup & Installation

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.
